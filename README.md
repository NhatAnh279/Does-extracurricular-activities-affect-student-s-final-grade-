# Do extracurricular activities affect grades?

## Goal
Analyze whether extracurricular activities are associated with students' final grades.

## Approach 
Using statistic to decide whether extracurricular activities affect final grade and linear regression to estimate the effect of extracurricular activities while controlling for other multiple factors such as studytime, absences...

## Key Result
Extracurricular activities show a small positive association with final grades, but the effect is modest compared to core academic predictors.

## Interpretation
The findings suggest extracurricular activities may support academic performance but are not the primary determinant of student success.

## Limitation
This analysis is observational and does not establish causality.

## Tools
Python, SQL, jupyter
